# Installation
### to install just clone the repo like this:
### git clone https://github.com/Evangelospro/ZSHRC ~/.zsh
### and run ~/.zsh/install.sh
## Oneliner:
### git clone https://github.com/Evangelospro/ZSHRC ~/.zsh && ~/.zsh/install.sh
